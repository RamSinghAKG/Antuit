# Antuit
Project Setup steps:
1) git clone https://github.com/RamSinghAKG/Antuit.git
2) cd server
3) npm install
4) npm start   // will start server -->   http://localhost:9000 
5) cd client
6) npm install
7) npm start    // open browser: http://localhost:3000
